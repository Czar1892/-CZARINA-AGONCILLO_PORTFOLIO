## ğŸ¨ Customization

### Colors

Edit the CSS variables in the `<style>` section:

```css
:root {
    --primary: #116a93;
    --primary-dark: #0d5373;
    --primary-light: #1a8bc4;
    /* ... more variables */

```

### Content

All content can be edited directly in the HTML. Look for sections with IDs:
- `#home` - Hero section
- `#skills` - Skills section
- `#projects` - Projects section
- `#testimonials` - Testimonials section
- `#contact` - Contact section

### Fonts

Current fonts are loaded from Google Fonts. To change them, update the link in the `<head>`:

```html
<link href="https://fonts.googleapis.com/css2?family=Your+Font&display=swap" rel="stylesheet">
```
